## 📌 소개

**잡으숏** 은 기존 서류 중심의 채용 방식에서 벗어나,  
**숏폼 영상으로 지원자의 태도·말투·역량을 빠르게 파악**할 수 있는  
**청년 친화형·지역 기반 채용 플랫폼**입니다.

틱톡/인스타 릴스/유튜브 숏츠처럼 영상을 스크롤하며  
지원자·기업 간의 **빠르고 직관적인 연결**을 제공합니다.


## 🎯 문제 정의

기존 채용 시장의 구조적 한계:

- 서류 중심 → 사람의 태도/커뮤니케이션이 보이지 않음  
- 지역 청년 & 지역 기업 간 연결 부재  
- 중소기업의 **이력서 홍수**로 인한 검토 피로  
- 청년의 **반복 자기소개서 작성 부담**  

> 잡으숏은 이 문제를 해결하기 위한 **영상 기반 채용 플랫폼**입니다.


## 🚀 주요 기능

### 지원자용
- **1회 촬영 프로필 영상** 업로드  
- AI 기반 자동 요약 & 키워드 태깅  
- 지역 기반 추천  
- 기존 영상 재사용 가능  

### 기업용
- 숏폼 영상 스크롤로 지원자 탐색  
- 관심 후보자 저장  
- 즉시 채팅 및 인터뷰 요청  
- 직무·지역·경력 기반 AI 추천  

### 상호작용 & 확장성
- 기업 ↔ 지원자 **실시간 메시지**  
- 기업 소개 숏츠 영상 제공  
- 지역 행사·박람회 연동 가능  
- 지산학 협력 & 교육기관 연계 가능

## 📈 기대 효과

### 기업
- 이력서 검토 시간 **주당 6~8시간 절감**  
- 영상으로 태도/말투 판단 가능 → **잘못된 채용 감소**  
- 지역 인재 확보 쉬움  

### 지원자
- 자기소개서 반복 작성 X → **1회 촬영으로 지속 활용**  
- 자신의 말투·표현 등 **소프트 스킬 어필 가능**  
- 하나의 영상 → 여러 기업에 자동 노출  

### 지역 사회
- 지역 기업·청년 연결 강화  
- 중소기업 인력 부족 문제 완화  
- 지산학 데이터 선순환 구조 형성
